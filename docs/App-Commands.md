## Firefox

| Command            | Action                | Note |
| ------------------ | --------------------- | ---- |
| `Fox <Left/Right>` | Go to Left/Right tab  |      |
| `Fox <Down/Raise>` | Page Down/Up          |      |
| `Fox Fish`         | Press Ctrl+F          |      |
| `Fox <Num>`        | Switch to tab <Num\>  |      |
| `Fox Limo`         | Open link in new tab  |      |
| `Meta`             | Close current tab     |      |
| `Duke`             | Duplicate current tab |      |
